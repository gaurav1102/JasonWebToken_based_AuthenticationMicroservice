# API Examples

## Signup

```json
{
  "username": "gaurav123",
  "email": "gaurav@example.com",
  "password": "Password1"
}
```

## Login

```json
{
  "username": "demo_user",
  "password": "Password1"
}
```

## Refresh Token

```json
{
  "token": "<refresh-token>"
}
```

## Logout

```json
{
  "refreshToken": "<refresh-token>"
}
```
